# SAT SOLVER-Implementation


> le code contien un solver d'instance en version C++ et une version python le code python fait lealalyse complete de chaqye heuritique et aussi donne resultat et aussi dessin de graph.
> le C++ en revange se consentre le solver lui meme uniquement  utilisant la technique NAVIVE , l'heuristique MOMS et CDCL , tendit que les code comparaison.py et instanceGenerator.py sont la pour completer le code de C++ , instance generator sert a cree les instance de test (en forma .cnf) , et comparaison , a en entré la sortie du terminal du code C++ apres execution , et donne les graph comparatif 
---

## Fichiers important présents dans le docier Res :

- **`.cnf`** : les script sous forma DINAMX contenant la discription de l'instance elle meme 

- **`.cnf.sol`** : la solution si SAT donné par le code C++.

---
## NOTE : si le le Solver C++ ne trouve pas de solution ( UNSAT) il ne générera pas de fichier cnf.sol, mais uniquement le temps d'execution et les noeux explorés
